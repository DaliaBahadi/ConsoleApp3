# ConsoleApp3

The Product Inventory app is used to add a product with its name, unit price, and quantity to a products list.

### **Technologies used:** 

- Visual Studio Code (Version: 1.40.1)
- .Net Core framework(Version: 3.0 SDK)
-  Blazor Server App


### **Programming languages:**

- C#   (Version: 8)
- HTML (Version: 5)
- CSS  (Version: 3)
- Bootstrap (Version: 4)
- Razor
